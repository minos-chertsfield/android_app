# "垂涎"美食App
![logo](https://github.com/minos-chertsfield/android_app/blob/master/app/src/main/res/mipmap-mdpi/logo96.png "垂涎")

### 一、注册功能
#### &emsp;&emsp;界面中的文本控件可获取文本，其中有提示文本，日期可通过点击调取万年历控件进行选择。头像控件可触发摄像头或调取用户本地相册上传头像。界面提供用户协议文本，用户阅读后勾选同意，可进行注册。

<img src="https://github.com/minos-chertsfield/android_app/blob/master/app/src/main/res/drawable/img1.jpg" width=240 height=400/>

### 二、登录功能
#### &emsp;&emsp;界面背景采用淡橘色，登录的文本输入框采用凹陷效果，其中同样有文字提示。
<img src="https://github.com/minos-chertsfield/android_app/blob/master/app/src/main/res/drawable/img2.jpg" width=240 height=400/>

### 三、功能主页
#### &emsp;&emsp;App的首页，用于展示一些重要的信息，并为其他功能页面提供导航。
<img src="https://github.com/minos-chertsfield/android_app/blob/master/app/src/main/res/drawable/img3.jpg" width=240 height=400/>

### 四、课程功能
#### &emsp;&emsp;界面将课程预览图与名称结合，组合成条目，并且采用控件中的卡片布局，进行倒角，并设置阴影效果。
<img src="https://github.com/minos-chertsfield/android_app/blob/master/app/src/main/res/drawable/img4.jpg" width=240 height=400/>

### 五、商城功能
#### &emsp;&emsp;当前页面中，将卡片布局绑定到交错布局管理器上，并且为每个条目的高度绑定随机数，让条目高度能够在刷新时变化。每个条目上显示商品的预览图、名称以及对应价格。
<img src="https://github.com/minos-chertsfield/android_app/blob/master/app/src/main/res/drawable/img5.jpg" width=240 height=400/>

### 六、菜谱功能
#### &emsp;&emsp;用户可以在界面中根据提示添加图片和文字，通过按钮可以动态增加控件来添加菜谱的步骤。
<img src="https://github.com/minos-chertsfield/android_app/blob/master/app/src/main/res/drawable/img6.jpg" width=240 height=400/>
#### &emsp;&emsp;界面中显示菜谱的封面以及作者的头像和用户名。     
<img src="https://github.com/minos-chertsfield/android_app/blob/master/app/src/main/res/drawable/img7.jpg" width=240 height=400/>

### 七、个人信息
#### &emsp;&emsp;用ListView方式显示用户个人信息。
<img src="https://github.com/minos-chertsfield/android_app/blob/master/app/src/main/res/drawable/img12.jpg" width=240 height=400/>

### 八、设置功能
#### &emsp;&emsp;常见的的设置选项。

### 九、评论功能
#### &emsp;&emsp;将评论区嵌套在帧布局中，利用重写Adapter显示评论区布局，根据后台刷新数据动态显示，绑定Handler及时反馈数据。
<img src="https://github.com/minos-chertsfield/android_app/blob/master/app/src/main/res/drawable/img11.jpg" width=240 height=400/>

### 十、收藏功能
